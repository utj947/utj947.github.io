# 웹사이트 자동 업데이트 시간 표시 (Last Updated) 설정 안내

웹사이트 하단에 표시되는 **마지막 업데이트 시간**은 매번 수동으로 변경하거나 GitHub Actions와 같은 복잡한 서버 설정을 할 필요가 없습니다. 

현재 적용된 방식은 웹 브라우저가 화면을 그릴 때 **GitHub API를 활용해 이 레포지토리(utj947.github.io)의 최신 커밋(Commit) 시간을 실시간으로 불러와서 보여주는 방식**입니다.

### 어떻게 동작하나요?

`index.html` 파일 하단의 JavaScript 코드에서 아래와 같이 동작합니다.
1. `https://api.github.com/repos/utj947/utj947.github.io/commits/main` 주소로 요청을 보냅니다.
2. 깃허브에서 가장 최근 커밋된 날짜와 시간 정보를 받아옵니다.
3. 받아온 시간을 한국 시간 기준(ko-KR)으로 보기 좋게 변환합니다. (예: `2026. 06. 10. 15:30`)
4. 변환된 시간을 페이지 하단의 `<p id="last-updated">` 요소 안에 자동으로 삽입합니다.

### 유지 보수 방법

- 굳이 날짜를 수동으로 수정할 필요가 **전혀 없습니다.** 
- 코드를 수정하고 **`git push`**만 하면, GitHub API에 새로운 커밋 기록이 남게 되므로 사이트 방문자는 새로고침 시 항상 최신 업데이트 시간을 보게 됩니다.
- 브라우저 단에서 처리되기 때문에 별도의 서버 관리나 토큰이 필요하지 않습니다.
