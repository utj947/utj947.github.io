음력 생일을 구글 캘린더에 추가하기

Lunar2GoogleCalender: 음력 생일을 Google 캘린더에 간편하게 추가하고 매년 자동 업데이트 받으세요!

✨ 주요 기능

  • 음력 날짜를 양력으로 **정확하게 자동 변환**하여 Google 캘린더에 추가합니다.
  • 올해부터 **최대 50년**까지 자동으로 반복되는 생일 이벤트를 생성합니다.
  • 사용자 본인의 **Google 캘린더와 직접 연동**되어 안전하고 편리하게 생일을 관리할 수 있습니다.
  • 복잡한 음력-양력 변환 없이 간편하게 소중한 사람들의 생일을 놓치지 않을 수 있습니다.

🚀 사용 방법

  1. 크롬 웹 스토어에서 "음력 생일을 구글 캘린더에 추가하기" 확장 프로그램을 설치합니다. (아직 배포 전)
  2. 브라우저 툴바에 있는 확장 프로그램 아이콘을 클릭하여 팝업 창을 엽니다.
  3. 팝업 창에 음력 월, 음력 일, 이벤트 이름(예: "엄마 생신"), 그리고 몇 년간 추가할지 입력합니다.
  4. '구글 캘린더에 추가하기' 버튼을 클릭합니다.
  5. (최초 사용 시) Google 로그인 및 캘린더 접근 권한 요청 동의 절차를 완료합니다.
  6. 입력된 음력 생일이 각 년도의 양력 날짜로 변환되어 당신의 Google 캘린더에 자동으로 추가됩니다.
